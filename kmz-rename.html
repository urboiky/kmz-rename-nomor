Rename Placemark di KMZ

Aturan: hapus prefix No/No. (case-insensitive), hilangkan tanda baca di depan (.,;:). Jika hasilnya angka murni 1–9 → jadi 01–09. Jika ada huruf/simbol (mis. 4b, 7/1b, b10) → tidak dipadding, tapi huruf jadi UPPERCASE. Popup foto/HTML di <description> dan file gambar di KMZ tetap.
